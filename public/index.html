pillars

multimodal

Configure chat, reasoning, and OCR models independently. Switch providers per task instead of being locked into one.

human in the loop

Write actions pause for explicit approval. The agent shows a preview and waits for Accept or Refuse before executing.

craftable modules

Build connector packages with custom tools, auth, prompts, and previews. Verticalize the framework for your domain or company.

memory management

User memory is authoritative. Create project contexts to scope work and switch between them instantly.

compression

Verbose tool output is compressed before it re-enters the model context, keeping responses fast and focused.
